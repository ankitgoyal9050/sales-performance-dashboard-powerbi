
# 📊 Sales Performance Dashboard | Power BI

## 📌 Project Overview

This project is an interactive Sales Performance Dashboard built in Power BI using a Star Schema Data Model.

The dashboard helps analyze business performance through Sales, Profit, Quantity, Monthly Profit Trend, Brand Performance, Category Analysis, and interactive filters.

---

## 🎯 Business Objective

The objective of this dashboard is to help management:

- Track Total Sales
- Monitor Total Profit
- Analyze Sales Quantity
- Compare Monthly Profit Trend
- Identify Top Performing Brands
- Analyze Product Categories
- Filter reports by Region, Category, Gender, and Year

## 📌 Key Insights

- Dell generated the highest profit.
- Furniture contributed the highest sales.
- Monthly profit trend allows comparison between 2024 and 2025.
- Dashboard provides interactive analysis using slicers.
- Business performance can be analyzed by Brand, Category and Region.
